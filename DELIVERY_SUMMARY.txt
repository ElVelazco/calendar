### #1 Constants Extraction
**Impact**: Theme changes from 5-10 minutes → 2 minutes
```python

```

### #2 Duplicate Code Elimination
**Impact**: -150-200 lines of code
```python
# Treeview setup: Was in 2 places, now in 1 method
def _create_treeview(self, parent, tree_type): ...

# Used twice:
self.tree = self._create_treeview(container, "calendar_list")
self.manage_tree = self._create_treeview(container, "manage_list")
```

### #3 Helper Functions
**Impact**: Tag extraction from 4 places → 1 place
```python
def _extract_event_idx_from_tree(self, tree_widget, item_id):
    """Extract event index from tree item tags."""
    # ... centralized logic ...
```

### #4 Error Handling
**Impact**: From bare except → specific error types
```python
# BEFORE: except:
# AFTER: except (IOError, ValueError, json.JSONDecodeError) as e:
```
